Loconomics Style Guide

Based on Bootstrap customizing basics as colors and typography, extended with our own components.

Overview

Bootstrap

This site is using a custom build of Bootstrap v3.2.0, following this styleguide values for customized colors and more.

It uses too the set of styles and helper classes of the Style Guide page from getbootstrap.com, with some minor changes.

Typography

Mobile

We use the standard fonts used by Bootstrap, they are:

  • @font-family-sans-serif "Helvetica Neue", Helvetica, Arial, sans-serif:
    This is the @font-family-base, applies by default to all the text (body copy).
  • @font-family-serif Georgia, "Times New Roman", Times, serif:
    Alternative when a Serif font is wanted (useful for printing), not used by default.
  • @font-family-monospace Menlo, Monaco, Consolas, "Courier New", monospace:
    Alternative when a Monospace font is wanted, used for blocks of code and preformatted text.

Desktop

Only on desktop, we use downloaded web fonts for the site, with some fallbacks for the minor cases where formats WOFF, TrueType, SVG and Embedded-OpenType are not supported.

  • Alte Haas Grotesk (CSS fallbacks: Helvetica, Arial, sans-serif), as in following sample:
    Alte Haas Grotesk is the main typography, applies by default to all the text (body copy).

    Hand Of Sean is used for the higher titles only.
