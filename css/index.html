CSS Components

Visual components, custom elements made only of CSS.

All are usable as is, but some could get enhanced using a complementary Javascript component.

Overview

Help Point

A small marker to link a help page.

Inline help

To use in-line help through Popovers the Javascript component counterpart is required.

Basic example

This component is meant to be used with a short text, mainly a question mark (?). Just use the HelpPoint class in a link.

Sizing

The size of the Help Point is relative to its container (em units) so the default size adjust to the line size of the in-line text.

To decrease or increase the size use the size-* classes.

Positioning

By default the HelpPoint goes inline with the text as any other text or link.

To place it on the right side of its container, like for example when used with a title, set the has-HelpPoint class to the container element.

Relative positioning

Because the has-HelpPoint changes the element position to be relative, this may cause conflicts with some layouts where static positioning is explicitely used in the parent element, that cases may need an extra container where to set the class without side effects.

Padding

The Help Point is placed on the right using absolute positioning and a value of right: 0, causing that any padding applied to the container has no effect to the it, something unwanted on some cases. To give it a padding, put a margin-right on the container or overlap the default value of right in the Help Point.
