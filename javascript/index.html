Javascript Components

Enhanced and reusable UI/UX components.

Some of them are built on top of CSS Components that are also usable independently.

Overview

Component data attributes

Don't use data attributes from multiple plugins on the same element. For example, a button cannot both have a tooltip and toggle a modal. To accomplish this, use a wrapping element.

Help Point

A small marker with a Popover for inline help.

Display options

Sizes and other pure CSS options for the marker are described in the CSS Component counterpart

Basic example

This component is meant to be used with a short text that triggers a Popover on click to provide in-context help.

Just use the HelpPoint class in a link, recommended with href="#", and the attribute data-content with the desired text.
