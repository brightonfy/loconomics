Javascript Components

Enhanced and reusable UI/UX components.

Some of them are built on top of independent CSS Components.

Overview

Component data attributes

Don't use data attributes from multiple plugins on the same element. For example, a button cannot both have a tooltip and toggle a modal. To accomplish this, use a wrapping element.

Help Point

A small marker with a Popover for inline help.

Display options

Sizes and other pure CSS options for the marker are described in the its complementary CSS Component

Basic example

This component is meant to be used with a short text that triggers a Popover on click to provide in-context help.

Just use the HelpPoint class in a link, recommended with href="#", and the attribute data-content with the desired text.

Carousel

The Bootstrap Carousel component is available with all its options as described in official Bootstrap docs

It's mainly used to expose images, that fit all the Carousel size, with an optional caption over the image.

In order to be used with text that flows from the top (and not as a centered caption), an optional sub-element was introduced carousel-content, placed inside a container on each item, allows for text, or any formatted content, as replacement for the single exposed image.
